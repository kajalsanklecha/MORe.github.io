Interactive Confusion Matrix Instructions and Details

  • Hovering on the individual elements of the matrix shows its actual and predicted classes and the average values across all the objects of that type.
    • "x": The Predicted Class
    • "y": The Actual Class
    • "color": The average value of all the elements with the given actual and predicted class.
  • Hover over the buttons on the top right to know their functions.

Interactive Exploration of tSNE: Instructions and Details

  • Hovering on individual elements in the scatter plot show details of individual models in the new 2D tSNE space.
    • "l": Class of the specified object.
    • "x": x-coordinate for tSNE plot of the element.
    • "y": y-coordinate for tSNE plot of the element.
    • "f": Name of the element in the ModelNet40 dataset.
  • Single click on any active class in the right panel removes that class from the current view.
  • Double click on any class shows only that class in the current view.
  • Single click on any inactive class adds that class to the current view.
  • To reset the view to "All classes active", double click in the background of the right panel.
  • Hover over the buttons on the top right to know their functions.

Interactive version of the tSNE visualization shown in Figure 6 in the main paper. It shows the 2D tSNE visualization of the embeddings of all the models taken from ModelNet40.
